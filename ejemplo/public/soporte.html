Costos y Licencias

Nuestra solución de agendas médicas se ofrece bajo un modelo de licenciamiento flexible, adaptable a las necesidades de su práctica o institución. Ofrecemos diferentes planes que incluyen:

  • **Licencia de Uso:** Tarifa mensual o anual basada en el número de usuarios o sedes.
  • **Personalización y Desarrollo:** Costos adicionales por funcionalidades específicas o integraciones a medida.
  • **Soporte y Mantenimiento:** Paquetes de soporte técnico y actualizaciones continuas para garantizar el óptimo rendimiento.

Tecnologías de Código Abierto (Open Source): La Base Sólida de Nuestro Proyecto

En el corazón de nuestra solución de agendas médicas late un compromiso con la innovación, la eficiencia y la reducción de costos. Para lograrlo, nos apoyamos en un stack de tecnologías de código abierto, robustas y de probada eficacia, que nos permiten construir un sistema de alta calidad sin incurrir en onerosas licencias de software propietario.

React

Biblioteca de JavaScript para construir interfaces de usuario interactivas y modulares. Permite crear componentes reutilizables, como la barra de navegación, el menú lateral, y las vistas de acarreos activos e historial, asegurando una navegación dinámica y actualizaciones en tiempo real.

Tailwind CSS

Framework de CSS basado en utilidades que agiliza el diseño de interfaces responsivas y modernas. Se utiliza para estilizar elementos como contenedores, márgenes, y botones, proporcionando un diseño consistente y adaptable a diferentes dispositivos.

Express

Framework de Node.js para desarrollar la API del servidor. Maneja solicitudes desde el frontend, como la obtención de acarreos activos, la cancelación de acarreos, y la gestión de usuarios, conectando la aplicación con la base de datos.

MySQL

Sistema de gestión de bases de datos relacional que almacena información sobre acarreos, usuarios, y notificaciones. Garantiza un acceso rápido y seguro a los datos, como el estado de los acarreos y el historial de estos mismos.

Node.js

Entorno de ejecución que permite correr el servidor backend, soportando la API de Express y la comunicación con MySQL.

Axios

Biblioteca para realizar solicitudes HTTP desde el frontend al backend, utilizada para obtener datos de mudanzas, actualizar estados, y manejar cancelaciones.

SweetAlert 2

Herramienta para mostrar notificaciones visuales al usuario, como confirmaciones de acarreos aceptados, conductores en camino, o cancelaciones exitosas, mejorando la experiencia de usuario.

React Router

Biblioteca para gestionar la navegación entre vistas, como "Inicio", "Perfil", "Histórico", y "Encuesta", sin recargar la página.

LocalStorage

API del navegador para almacenar datos persistentes, como el ID del usuario y las notificaciones vistas, optimizando la experiencia sin cargas innecesarias.

JavaScript (ES6 +)

Lenguaje principal que soporta toda la lógica de la aplicación, utilizando características modernas para un código eficiente y mantenible.

Vite

Vite es una herramienta de construcción y desarrollo frontend de última generación, diseñada para ofrecer un entorno de desarrollo rápido y eficiente para aplicaciones web modernas. Utiliza módulos ES nativos (ESM) del navegador para servir código sin necesidad de empaquetarlo durante el desarrollo, y genera builds optimizados para producción con Rollup. Vite es especialmente popular en proyectos con frameworks como React debido a su velocidad y configuración sencilla.

La adopción de estas tecnologías de código abierto no solo nos permite desarrollar soluciones de alta calidad, sino que también se traduce en una **reducción significativa de los costos operativos a largo plazo** para nuestros clientes, al no requerir el pago de licencias por su uso.
